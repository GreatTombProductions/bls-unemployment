# County Unemployment Check

Search county-level unemployment rates across the US. Monthly data from the Bureau of Labor Statistics Local Area Unemployment Statistics (LAUS) program.

**Currently covering 225 counties across 11 states.** Expanding to all 3,143 US counties.

## Features
- **County name search** — find unemployment rates by county
- **State browse** — see all counties in a state ranked by unemployment rate
- **County detail** — monthly trend chart, state/national comparison, year-over-year change

## Data
Bureau of Labor Statistics, Local Area Unemployment Statistics (LAUS), via public API.
Monthly rates, not seasonally adjusted. Latest month is preliminary and subject to revision.
